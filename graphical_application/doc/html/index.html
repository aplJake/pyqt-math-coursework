Example of working process of the software

Short description:

The purpose of project is design and implementation of a program that finds roots of an equation
f(x) = g(x) where f(x) is the polynomial of n-th power. The values of g(x) are assigned
using a set of points.Intermediate values should be calculated using Lagrange polynomial.

The program implement the following key functions:

– reading source data from the XML document
– editing and storing data in another XML document
– solving a given problem, with graphical representation of results
– report generation in HTML format.
