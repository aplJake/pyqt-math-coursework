GUI Description

How to solve the equation

User input

User can solve the equation by entering the data into the text fields under f(x) and g(x) tables and bounds fields.
When all necessary data is entered than user press Build button and gets the result in graphical and text form.
The graph window has 3 functions: f(x) g(x) and f(x)-g(x) with the solution matched with the points.

Input from file

User can solve the equation in file form.

To do this user have to prepare the .xml file on his own or just open
already saved .xml files (detailed description how to open .xml file below)

How user can save the equation data

XML Saving

If the user wants to save the entered data with solution and use it one more time, it can be done by saving
the equation in .xml file

HTML Saving

If the user wants to save the entered data with solution in HTML for .xml file

Editting equation

Build

User can also build the function with the usage of menu bar Build item

Clear

Uses if user wants to delete all the data about the equation
