can be calculated with the method of interpolation by Lagrange.

Solving process

When user entered data in f(x) and g(x) table and added the bounds from to EquationSolver solves the
equation on the interval that intersects user bounds and bounds of the function g(x) where this function is
defined. The solution proceded on the calculated interval with the usage of dichotomy method.
